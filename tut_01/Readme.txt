1.Download zip file from Github.
2.Unzip the file .
3.Open the file in code editor(eg. VS code).
4.Create Virtual environment in the terminal (python -m venv venv).
5.Activate venv (./venv/Scripts/activate.bat).
6.install all required packages from requirement.txt file(pip install -r requirements.txt).
7.Run the code through streamlit (streamlit run tut01.py).
8.Drag and drop or upload an input file.
9.Put the number of groups you want to create.
10.Click on the create group button.
11.Select one option out of 3 created option for the group type.
12.click on the created groups to download the output files.
